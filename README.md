# SMART-ATHLETE-CARD-LEAGUE-SIMULATION

Smart League Card Game is a desktop card game project developed in Python.  
The application simulates a sports-themed card match system where the player competes against the computer, cards are compared according to their attributes, and scores are calculated dynamically during the game.

This project was developed as a course/project assignment and includes game logic, data management, card visuals, and a graphical user interface.

---

## Features

- Sports card based gameplay
- Player vs Computer match system
- Dynamic score calculation
- Turn-based game flow
- Difficulty and game settings support
- CSV-based player/card data reading
- Graphical desktop interface
- End-of-match summary and result screen
- Unit tests for core components

---

## Technologies Used

- *Python*
- *PyQt / PyQt-based GUI structure*
- *CSV data processing*
- *Object-Oriented Programming (OOP)*
- *unittest*
